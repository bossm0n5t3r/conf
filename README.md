## Usage

### .vimrc

```sh
$ cd .vim
$ ln -sf ~/gitFolders/dotfiles/.vim/vimrc vimrc
$ vim vimrc

# Run :PlugInstall
```

### .tmux.conf

```sh
$ cd ~
$ ln -sf ~/gitFolders/dotfiles/.tmux.conf .tmux.conf
```
